# 第1节：个人信息 (Personal Information)

## 页面标题
个人信息

## 页面描述
注意：此页面上的数据必须与护照上的信息完全匹配。

---

## 字段列表

### 1. 姓（姓）/ Surname
- **字段ID**: `data.surname`
- **字段类型**: 文本输入框 (text input)
- **必填**: 是 (标记 *)
- **布局**: 左侧列，第1个字段
- **说明**: 请完全按照护照上的姓名输入所有姓氏（或姓氏）。如果您的护照上只写有一个名字，请输入该名字作为您的"姓氏"（例如FERNANDEZ GARCIA）

### 2. 名字（名字）/ Given Names
- **字段ID**: `data.given_name`
- **字段类型**: 文本输入框 (text input)
- **必填**: 是 (标记 *)
- **布局**: 左侧列，第2个字段
- **说明**: （例如，JUAN MIGUEL），如果您的护照中没有名字或名字，请在"给定名字"空格中输入" FNU"（意思是"未知名字"）

### 3. 本国文字姓名 / Name in Native Alphabet
- **字段类型**: 文本输入框 (text input)
- **必填**: 是 (标记 *)
- **布局**: 左侧列，第3个字段
- **配套复选框**: 两个复选框（二选一）：
  - `不适用` (Not Applicable) — 勾选后禁用文本输入框
- **说明**: 如果您本国的文字使用非罗马字母（如中文、日文、韩文、阿拉伯文等），请在此处用本国文字输入您的姓名。

### 4. 您是否使用过其他名称（例如，处女，宗教，专业，别名等）？
- **字段类型**: 单选按钮组 (radio group)
- **必填**: 是 (标记 *)
- **布局**: 左侧列，第4个字段
- **选项值**:
  - `true` - 有
  - `false` - 没有
- **衍生字段**: 当选择"有"(true)时，显示以下两个字段：
  - 使用的其他姓氏（未婚，宗教，专业，别名等）- 文本输入框 (ID: `data.used_other_name.surname`)
  - 使用的其他给定名称 - 文本输入框 (ID: `data.used_other_name.given_name`)

### 5. 您有代表您名字的电码吗？
- **字段类型**: 单选按钮组 (radio group)
- **必填**: 是 (标记 *)
- **布局**: 左侧列，第5个字段
- **选项值**:
  - `true` - 有
  - `false` - 没有
- **衍生字段**: 当选择"有"(true)时，显示以下两个字段：
  - 使用的其他给定名称 - 文本输入框，非必填
  - 请提供您的电码给定名称 - 文本输入框，必填 (*)
- **说明**: 电码是4位数字，代表某些非罗马字母名称中的字符。

### 6. 性别 / Sex
- **字段类型**: 单选按钮组 (radio group)
- **必填**: 是 (标记 *)
- **布局**: 左侧列，第6个字段
- **选项值**:
  - `M` - 男
  - `F` - 女

### 7. 婚姻状况 / Marital Status
- **字段类型**: 单选按钮组 (radio group)
- **必填**: 是 (标记 *)
- **布局**: 左侧列，第7个字段
- **选项值**:
  - `M` - 已婚
  - `C` - 普通法婚姻
  - `P` - 民事结合/家庭伴侣
  - `S` - 单身
  - `W` - 丧偶
  - `D` - 离婚
  - `L` - 合法分居
  - `O` - 其他
- **衍生字段**: 当选择"其他"(O)时，显示一个字段：
  - 婚姻状况说明 - 文本域 (textarea)，必填 (*)

### 8. 出生日期 / Date of Birth
- **字段类型**: 日期选择器 (3个下拉框组合)
- **必填**: 是 (标记 *)
- **布局**: 左侧列，第8个字段
- **子字段**:
  - **DAY** (ID: `data.date_birth_v2.DD`) - 下拉选择框，选项：01-31
  - **MONTH** (ID: `data.date_birth_v2.MMM`) - 下拉选择框，选项：一月、二月、三月、四月、五月、六月、七月、八月、九月、十月、十一月、十二月
  - **YEAR** (ID: `data.date_birth_v2.YYYY`) - 文本输入框，4位数字

### 9. 出生城市 / City of Birth
- **字段ID**: `data.place_of_birth.city`
- **字段类型**: 文本输入框 (text input)
- **必填**: 是 (标记 *)
- **布局**: 左侧列，第9个字段

### 10. 省份/出生州 / State of Birth
- **字段ID**: `data.place_of_birth.state`
- **字段类型**: 文本输入框 (text input)
- **必填**: 否
- **布局**: 左侧列，第10个字段

### 11. 出生国家 / Country of Birth
- **字段ID**: `data.place_of_birth.country`
- **字段类型**: 下拉选择框 (select dropdown, Ant Design Select)
- **必填**: 是 (标记 *)
- **布局**: 左侧列，第11个字段
- **选项**: 约280个国家/地区，中文显示，例如：阿富汗、阿尔巴尼亚、阿尔及利亚、美属萨摩亚、安道尔、安哥拉、中国、美国等

### 12. 原籍国家/地区（民族）/ Nationality
- **字段ID**: `data.nationality`
- **字段类型**: 下拉选择框 (select dropdown, Ant Design Select)
- **必填**: 是 (标记 *)
- **布局**: 右侧列，第1个字段
- **选项**: 约280个国家/地区，中文显示

### 13. 您是否拥有或拥有上述国籍以外的任何国籍？
- **字段类型**: 单选按钮组 (radio group)
- **必填**: 是 (标记 *)
- **布局**: 右侧列，第2个字段
- **选项值**:
  - `true` - 有
  - `false` - 没有
- **衍生字段**: 当选择"有"(true)时，显示以下字段：
  - 其他原籍国家/地区（民族）- 下拉选择框 (combobox)，必填 (*)，约280个选项
  - 您是否持有上述其他原籍国家/地区（国籍）的护照？- 单选按钮组 (有/没有)，必填 (*)
    - **嵌套衍生字段**: 当此子字段选择"有"时，显示：
      - 护照号码 - 文本输入框，必填 (*)

### 14. 您是否是上述来源国家/地区（国籍）以外的国家/地区的永久居民？
- **字段类型**: 单选按钮组 (radio group)
- **必填**: 是 (标记 *)
- **布局**: 右侧列，第3个字段
- **选项值**:
  - `true` - 有
  - `false` - 没有
- **衍生字段**: 当选择"有"(true)时，显示：
  - 其他永久居民国家/地区 - 下拉选择框 (combobox)，非必填

## 子标题：文件资料 (Documents)

### 15. 身份证号码 / National ID Number
- **字段ID**: `data.national_id_number`
- **字段类型**: 文本输入框 (text input)
- **必填**: 否
- **布局**: 右侧列，第4个字段
- **说明**: 您的国民身份证号是政府提供的唯一号码。美国政府为寻求工作的人（社会安全号码）或缴税的人（纳税人ID）提供唯一的号码。如果您没有这些数字，请留空

### 16. 美国社会安全号码 / U.S. Social Security Number
- **字段ID**: `data.social_security_number`
- **字段类型**: 文本输入框 (text input)
- **必填**: 否
- **布局**: 右侧列，第5个字段
- **说明**: 我们的国民身份证号是贵国政府提供的唯一号码。美国政府向寻求工作的人（社会安全号码）或缴税的人（纳税人ID）提供唯一的号码。如果您没有这些数字，请留空

### 17. 美国税号 / U.S. Taxpayer ID Number
- **字段ID**: `data.tax_id_number`
- **字段类型**: 文本输入框 (text input)
- **必填**: 否
- **布局**: 右侧列，第6个字段
- **说明**: 如果您没有这些数字，请留空

## 字段交互逻辑

### 条件字段显示规则

1. **本国文字姓名**:
   - 触发条件: 始终显示
   - 复选框"不适用"或"技术不可用"勾选后，文本输入框被禁用
   - 两个复选框互斥，勾选一个会自动取消另一个

2. **其他名称问题**:
   - 触发条件: "您是否使用过其他名称" 选择 "有"(true)
   - 衍生字段:
     - "使用的其他姓氏（未婚，宗教，专业，别名等）" - 文本输入框，必填
     - "使用的其他给定名称" - 文本输入框，必填

2. **电码问题**:
   - 触发条件: "您有代表您名字的电码吗" 选择 "有"(true)
   - 衍生字段:
     - "使用的其他给定名称" - 文本输入框，非必填
     - "请提供您的电码给定名称" - 文本输入框，必填 (*)

3. **婚姻状况**:
   - 触发条件: "婚姻状况" 选择 "其他"(O)
   - 衍生字段:
     - "婚姻状况说明" - 文本域 (textarea)，必填 (*)

4. **其他国籍**:
   - 触发条件: "您是否拥有或拥有上述国籍以外的任何国籍" 选择 "有"(true)
   - 衍生字段:
     - "其他原籍国家/地区（民族）" - 下拉选择框，必填 (*)
     - "您是否持有上述其他原籍国家/地区（国籍）的护照？" - 单选按钮组 (有/没有)，必填 (*)
       - 嵌套: 选择"有"时显示 "护照号码" - 文本输入框，必填 (*)

5. **永久居民**:
   - 触发条件: "您是否是其他国家/地区的永久居民" 选择 "有"(true)
   - 衍生字段:
     - "其他永久居民国家/地区" - 下拉选择框，非必填

## 布局说明

表单采用两列布局：
- **左侧列**: 包含姓名、其他名称、电码、性别、婚姻状况、出生日期、出生地信息
- **右侧列**: 包含国籍、双重国籍、永久居民身份、身份证号、社会安全号、税号

---

## 技术细节

- 使用 Ant Design 组件库
- 单选按钮: `ant-radio-group` 和 `ant-radio-wrapper`
- 下拉选择框: `ant-select` 组件，支持搜索功能
- 文本输入框: `ant-input`
- 日期选择器: 3个 `ant-select` 组合（日、月、年）
- 表单验证: 必填字段标记 `*`，未填写时显示"这是必填栏"错误信息
