# 第4节：以前的美国旅行 (Previous US Travel)

## 页面标题
以前的美国旅行

---

## 字段列表

### 1. 您以前去过美国吗？
- **字段类型**: 单选按钮组 (radio group)
- **必填**: 是 (标记 *)
- **布局**: 第1个字段
- **选项值**:
  - `true` - 有
  - `false` - 没有
- **衍生字段**: 当选择"有"(true)时，显示最近访问的详细信息表单：
  - 到达日期 - 日期选择器 (日/月/年组合)
  - 停留时间 - 文本输入框
  - 请明确说明（时间单位）- 下拉选择框，选项：年、月、周、天、少于24小时
  - "加上另一个" 按钮 - 可添加最多5次访问记录

### 2. 您是否曾经持有美国驾照？
- **字段类型**: 单选按钮组 (radio group)
- **必填**: 是 (标记 *)
- **布局**: 第2个字段
- **选项值**:
  - `true` - 有
  - `false` - 没有
- **衍生字段**: 当选择"有"(true)时，显示：
  - 驾驶执照号码 (Driver's license number) - 文本输入框，必填
  - 驾驶执照状态 (Driver's license state) - 下拉选择框，必填，美国各州列表

### 3. 你曾经被核发过美国签证吗？
- **字段类型**: 单选按钮组 (radio group)
- **必填**: 是 (标记 *)
- **布局**: 第3个字段
- **选项值**:
  - `true` - 有
  - `false` - 没有
- **衍生字段**: 当选择"有"(true)时，显示以下字段：
  - 上次签证签发的日期 - 日期选择器 (日/月/年组合)，必填
  - 签证号码 (Visa number) - 文本输入框，必填
  - **6个嵌套单选按钮组**:
    1. 您是否正在申请相同类型的签证？(有/没有)
    2. 您是否在签发上述签证的国家或地区申请？(有/没有)
    3. 您进行了指纹扫描吗？(有/没有)
    4. 您的美国签证曾经丢失或被盗吗？(有/没有)
    5. 您是否曾经取消或撤销美国签证？(有/没有)

### 4. 您是否曾经被拒绝过美国签证，或者被拒绝过美国入境，或者在入境口岸撤回了您的入境申请？
- **字段类型**: 单选按钮组 (radio group)

- **衍生字段**: 当选择"有"(true)时，显示：
  - "请详细说明" - 多行文本框 (textarea)，必填

### 5. 您是否曾经被美国国土安全部通过旅行授权电子系统（ESTA）拒绝旅行授权？
- **字段类型**: 单选按钮组 (radio group)
- **必填**: 是 (标记 *)
- **布局**: 第5个字段
- **选项值**:
  - `true` - 有
  - `false` - 没有
- **衍生字段**: 当选择"有"(true)时，显示：
  - "请详细说明" - 多行文本框 (textarea)，必填

### 6. 有没有人代表您向美国公民及移民服务局填写移民请愿书？
- **字段类型**: 单选按钮组 (radio group)
- **必填**: 是 (标记 *)
- **布局**: 第6个字段
- **选项值**:
  - `true` - 有

## 字段交互逻辑

### 条件字段显示规则

1. **以前美国访问记录**:
   - 触发条件: "您以前去过美国吗" 选择 "有"(true)
   - 衍生字段: 到达日期 + 停留时间 + 时间单位 + "加上另一个"按钮（最多5次）
   - 每次访问记录包含：日期、停留时间、时间单位

2. **美国驾照信息**:
   - 触发条件: "您是否曾经持有美国驾照" 选择 "有"(true)
   - 衍生字段: 驾照号码 + 驾照州

3. **美国签证信息**:
   - 触发条件: "你曾经被核发过美国签证吗" 选择 "有"(true)
   - 衍生字段: 签证签发日期 + 签证号码 + 6个嵌套单选按钮组

4. **签证拒绝/入境拒绝说明**:
   - 触发条件: 单选按钮选择 "有"(true)
   - 衍生字段: 详细说明文本框（textarea）

5. **ESTA拒绝说明**:
   - 触发条件: 单选按钮选择 "有"(true)
   - 衍生字段: 详细说明文本框（textarea）

6. **移民请愿说明**:
   - 触发条件: 单选按钮选择 "有"(true)
   - 衍生字段: 详细说明文本框（textarea）

---

## 技术细节

- 全部为单选按钮组（有/没有）
- 共6个radio group，每组2个选项
- 字段值存储为字符串 "true" 或 "false"
- 选择"有"后显示对应的衍生字段
- 选择"没有"后不显示额外字段
- 签证信息部分最为复杂，包含6个嵌套单选按钮组
- 访问记录支持动态添加（最多5条）
- 表单验证在点击"下一个"时触发
